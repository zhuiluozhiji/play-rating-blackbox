## 8. 问题、解决方案、局限性与结论

把“问题解决”“局限性”“结论”合成最后一章。

### 8.1 遇到的问题与解决方案

建议用表格：

| 问题 | 表现 | 解决方案 |
|---|---|---|
| 树状问卷复杂 | 路径不固定 | 自动路径探索 |
| 类别不均衡 | `18+` 过多 | 少数类补采 + macro-F1 |
| 重复样本 | 随机采样重复 | response signature 去重 |
| 特征泄漏风险 | 结果页特征后验 | answer-only 消融 |
| 少数类不稳定 | `16+` 样本少 | CV + 谨慎解释 |

### 8.2 实验局限性

简明写 4 点：

- 类别仍不均衡
- surrogate model 不等于官方规则
- 少数类指标不稳定
- result-page features 不能用于严格提交前预测

### 8.3 结论

三点收束：

1. 自动化采集可以构建真实黑盒样本集。
2. 问卷答案对年龄分级具有强预测能力。
3. LightGBM 表现最佳，但模型解释仍应被视为经验性近似，而非官方规则。
